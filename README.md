# FreeSpace Open

[![Coverity](https://img.shields.io/coverity/scan/870.svg)](https://scan.coverity.com/projects/870)

An open-source continuation of FreeSpace 2, the classic 1999 space combat simulator.

## About

Thanks to Volition's generosity, we've continued development of FreeSpace 2 as FreeSpace Open, building graphical enhancements to bring the original games up to modern standards, telling new stories set in the FreeSpace universe and original worlds, and creating spectacular total conversions that let players fly in the beloved universes of Babylon 5, Battlestar Galactica, Wing Commander, and Star Wars.

### Features

- Linux and macOS support
- OpenGL rendering
- OpenAL audio
- OGG support
- Hardware transform, clipping, and lighting
- Easy mod installation via [Knossos](https://fsnebula.org/knossos/)

## Getting the Game

FreeSpace Open requires a retail installation of FreeSpace 2:
- [GOG.com](https://www.gog.com/game/freespace_2)
- [Steam](https://store.steampowered.com/app/273620/Freespace_2/)

After installing, use [Knossos](https://fsnebula.org/knossos/) to upgrade to FSO.

## Building

```bash
git submodule update --init --recursive
```

See the [Building](https://github.com/scp-fs2open/fs2open.github.com/wiki/Building) wiki page for full instructions.
